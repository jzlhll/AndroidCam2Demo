1. 这是一个android camera2 API的demo；
	通过使用设计模式，让复杂的camera2更加简单地操作：更容易让人理解camera2.
	本demo包含4种state模式，根据预览需求的session不同而创建。
	由2种View（surfaceView、TextureView）定制预览。
	MediaRecord录像。
	ImageReader拍照。
	基本涵盖了一般的需求功能。自认比Camera2Basic和Camera2Video简单，架构清晰。
2. 包含很多小工具类。
